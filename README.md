# VK ToadBot — Мультиаккаунтный бот для игры «Жабабот»

Мониторинг и автоматизация мультиаккаунтного бота игры «Жабабот» в ВКонтакте. Веб-панель для управления аккаунтами, отслеживания состояния жаб и распознавания ответов бота-игры.

---

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

Веб-панель доступна на `http://localhost:8000`.

---

## Структура проекта

```
src/
├── database/
│   └── db_manager.py          # Две БД (bot.db, recognition.db), схемы, посев, маппинг
├── vk/
│   ├── client_manager.py      # Управление пулом VK-аккаунтов (LongPoll)
│   └── handlers.py             # Обработчики сообщений, триггеры парсеров
├── utils/
│   ├── toad_info_parser.py    # Python-парсеры (Жаба инфо, Моя жаба, Мой инвентарь)
│   ├── knowledge_base.py      # Триггеры команд, match_bot_response
│   └── logger.py              # Логирование
├── web/
│   ├── server.py              # FastAPI эндпоинты (API, мониторинг, recognition)
│   └── static/
│       ├── app.js             # Веб-интерфейс (панель, статистика, аккордеон правил)
│       └── style.css          # Стили
config/
│   └── settings.py            # Настройки, лимиты, константы
data/
│   ├── bot.db                 # Аккаунты, настройки бота
│   └── recognition.db         # Состояния жаб, инвентарь, кулдауны
main.py                        # Точка входа
```
